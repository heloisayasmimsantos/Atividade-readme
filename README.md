# 📌 Número Secreto
## 📚 Descrição do Projeto
Este projeto foi criado para apremder a criar um readme.
O objetivo do sistema é para descobrir o numero secreto do jogo.
## 🚧 Status do Projeto
(X) Em desenvolvimento
( ) Finalizado
## 🔨 Funcionalidades
-Geração aleatória de um número secreto entre 1 e 100
-Sistema de dicas informando se o palpite é maior ou menor que o número secreto
-Contador de tentativas para acompanhar o desempenho do jogador
-Validação de entrada para aceitar apenas números
-Opção de reiniciar o jogo após acertar ou desistir
## 💻 Tecnologias Utilizadas
-HTML5 - Estrutura da página
-CSS3 - Estilização e layout responsivo
-JavaScript - Lógica do jogo e interatividade
-Git - Controle de versão
